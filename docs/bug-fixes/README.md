# Bug-fix register

> Parent: [documentation index](../README.md)

This directory preserves dated evidence for behavior, reliability, security,
and user-facing regressions. It is historical context, not living product
documentation.

Create `YYYY-MM-DD-<slug>.md` from [_template.md](./_template.md). Keep the
frontmatter fields `area`, `category`, and `severity`, then describe the symptom,
cause, fix, regression evidence, and remaining limits. Typo-only and formatting
changes do not need an entry.
