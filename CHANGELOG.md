# Changelog

All notable Trellis changes are recorded here. The repository follows semantic
versioning once public releases begin.

## 0.1.0 - 2026-07-11

- Established the dependency-free `trellis` CLI and safe project scaffold.
- Added canonical cross-agent skills with a generated Claude compatibility
  mirror and removed legacy prompt/command trees.
- Added the nine-phase SDD workflow, deterministic self-tests, documentation
  checks, configuration management, and honest gate result semantics.
- Added the read-only repository map plus opt-in Graphify, Bounds, and Phoenix
  integration contracts.
- Reworked public documentation, packaging, licensing, and clean-scaffold
  verification for the first source distribution.
