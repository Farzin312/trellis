# System documentation

> Parent: [documentation index](../README.md)

This directory contains living subsystem maps for this project. It starts with
no inferred subsystems; create one only when source ownership and public
boundaries can be verified.

Use the [subsystem template](../_subsystem-template.md). Each subsystem gets a
folder with a `README.md`, and this index links to it. Completed specifications
are historical evidence and do not replace the living map.
